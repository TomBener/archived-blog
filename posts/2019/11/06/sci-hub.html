# Why Sci-Hub?

今天我们来说一说 Sci-Hub。Sci-Hub 是什么？简单来说，Sci-Hub 是一个让你免费下载学术论文的网站，被称为海外免费的「破解版知网」[1]。

对于这个利器，曾几何时，我是看不上的。因为坐拥西南地区藏书规模最大的大学图书馆[2]，时常混迹于学校购买的各大数据库，还有什么文献是我找不到的[3]？于是以前在群里看见各种文献求助帖，我表示完全不能理解，居然还有文献是找不到的[4]？

不过事实证明，还是我太年轻了，确实有很多文献是很难获取到的。

2011 年，哈萨克斯坦研究生亚历珊卓·艾尔巴金（Alexandra Elbakyan）就因为这个问题而被困扰。她因为研究论文成本过高，每篇论文在付费墙机制（Paywall）下通常需要花费 30 美元，而决定建立 Sci-Hub。

2017 年，宾夕法尼亚大学的生物数据科学家 Daniel Himmelstein 研究了 Sci-Hub，发现该网站上的论文共有约 81,600,000 篇，包含了所有学术论文的 69%，基本能满足大部分的论文需求，因为剩下的 31% 是研究者不想获取的论文。今年 9 月 Sci-Hub 成立 8 周年之际，在 Twitter 上 宣布，其一共拥有 76,000,000 篇论文，每天约有 400,000 名用户访问。

显然，Sci-Hub 的存在严重侵犯了数据库厂商的利益，遭到了 Elsevier 等出版商的严厉 封杀。但是，随着加州大学 取消订购 Elsevier 出版的所有期刊，德国和瑞典的研究人员与 Elsevier 的 对抗，开放获取（OA, Open Access）的春天真的已经到来了吗？

然而事实是，Sci-Hub 经常变化的域名证明春天还远没有到来。正是由于 Sci-Hub 经常变化的域名，难以找到可用的网址，给很多人使用它带来了一定的困难。下面，我们就来盘点一下使用 Sci-Hub 下载文献的 5 种方法。

# Sci-Hub 使用方法盘点

# Sci-Hub Desktop

Sci-Hub Desktop[5] 是一款国人开发的 Sci-Hub 桌面版程序，输入所需文献的 DOI 号，即可一键获取 PDF 全文，即使在所有 sci-hub.xx 域名用不了的情况下也能正常下载很多文献的全文！Sci-Hub Desktop 只有 Windows 版本[6]，可以前往 这里 查看使用方法和下载地址。

# Where is Sci-Hub now?

顾名思义，Where is Sci-Hub now? 是一个告诉你现在哪一个 Sci-Hub 域名可用的网站。使用方法很简单：点击主页面的可用网址，跳转之后，在输入框内输入文献 DOI 号，回车即可获取 PDF 文件[7]，简直就是傻瓜式操作～

# Auto Sci-Hub 插件

Auto Sci-Hub 是一个浏览器扩展插件，支持 Firefox[8]。操作非常简单，甚至都不用搜索 DOI 号。当你浏览的网页中有论文下载地址，点击 Auto Sci-Hub 的小鸟图标，然后就会跳转到熟悉的 Sci-Hub 下载界面。可以前往 Firefox 扩展商店 下载 Auto Sci-Hub。

# 快捷指令

如果身边没有电脑，又急需下载论文，可以在 iPhone 或 iPad 上使用 快捷指令 来下载。点击这里 下载这个 SCI-HUB 快捷指令，然后按照下方动图中的方法使用[9]。

# 万能的 GitHub

最后，如果以上还不能满足你使用 Sci-Hub 的需求的话，不妨试试在万能的 GitHub 上搜索一下 Sci-Hub。无论是 Python 写的还是 JavaScript 写的，无论是图形化界面还是命令行，无论是 Chrome 扩展还是网页小书签（bookmarklet），应有尽有，满足你的各种需求。

当然，我觉得 GitHub 上这些项目对于使用 Sci-Hub 来说 过于极客，用 Kieran Healy 教授的话说叫做「overkill」。对我来说，前面的 4 种方法已经足够。话说回来，找文献只是最初级、最简单的一步，最重要的难道不是读文献吗？🌚
