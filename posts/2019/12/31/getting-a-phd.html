这篇文章为你取得博士学位提供了一个路线图，呈现在这里的，包括作者自己等各种来源渠道的观点。主要根据 M.H. Bellare 的《读博的经验》改编（已获得许可），同时对 Doherty 教授、Aydil 教授和 Chmelka 教授给出的建议和评论一并表示感谢。文中的观点不代表上述人士所在的大学或单位。

# 1. 你希望从读博中得到什么？

大部分学生认为读博就是要在科学领域解决一些科学问题，或者在特定的科学领域提出一种新方法。然而，这并不是读博的全部，下面是一些读博更具普遍性的目标。

  1. 你应该在理性思考的能力上更加有自信，并且将其运用于更加广泛的领域。生活本身就是一连串的问题组成的，我们有多经常这样看待生活，并有条不紊地解决我们的问题呢？博士毕业之后，你应该具备研究任何东西的意愿和能力，无论是化学工程、生物、烹调、园艺还是股票市场。
  2. 你应该有信心去质疑你周围的事物，并寻找新方法来弄明白他们。你应该倾向于问为什么事情要以某种方式来做，并试着把他们做得更好。重要的发现或发明往往来自于重要的问题。
  3. 博士学位应该能给你进入一个新领域的自信，让你迅速上手，并对这个领域有一些有趣的见解，即使其他人已经研究这个领域很长时间了。除了自己的博士论文领域的深度之外，它还会给你进入一个新领域的勇气。
  4. 你可能会在生活的各个方面增强对创造力的欣赏。你可能对艺术有不同的看法，或者对音乐有不同的看法。你可能会更加欣赏完成某些事情所付出的努力，以及新的发展如何与以前的工作相分离，以及学会重视创造力并去发现它。
  5. 读博将逐渐形成一种鉴赏力，它使你不愿意接受常见的标准，并让他们来测试你自己的智力和观点。你会很自然地发现自己在质疑一些事情。你应该愿意反驳传统观点，但这并不意味着仅仅是为了反抗而反抗，它只是一种建设性的批判。
  6. 博士学位应该能让你提高口头和书面的沟通技巧。它应该教会你与学识渊博的人交流你的想法。

# 2. 读博需要什么？

也许大多数学生进入科学研究的最普遍的愿景，是他们将必须解决一些前人没有解决过的难题。那将会是他们的论文，这也许是对的。但科研不仅仅是解决问题，它涉及概念化，发现问题，确定方向以及产生批判性见解。解决问题总是存在的，但是它所扮演的角色是多种多样的。一名成功的博士生应该教你回答问题，也教你提出重要的问题。 影响力大的答案的关键在于影响力大的问题。

博士学位论文的常见情形是提出一种新方法、新模型或新概念。当你拥有新的方法或模型时，第一批解决方案的技术方面可能非常简单，然后你可能想知道为什么没有其他人这样做。 原因是他们没有提出那个特定的问题或以特定的方式看待问题。不要看不起简单的东西，好的研究通常都非常简单。

大多数学生在攻读博士学位时，都认为自己喜欢科研，喜欢科研就是喜欢做它，而不是喜欢思考它把你放在什么地方，或者喜欢把自己看成是做科研的人，将研究过程与任何预期目标分开。如果你喜欢科研的过程，那么在过程中你只会感受到愉快，如果科研对你来说只是实现某些目标的手段，那么你就不太可能会喜欢它。记住，科研不是每天都有趣，也不是所有的科研都有趣，但我们大多数人都很享受这个过程。对大多数教授来说，思考一个科研问题不是他们的工作，但这是他们最喜欢做的事情之一。你可能不那么喜欢科研，但如果它只是你要完成的工作，你想着尽快摆脱，那你就入错行了。你仍然可能完成你的博士学位，但你可能不会有真正的成就感或幸福感。

# 3. 学会如何进行研究

把科研做得好，并没有诀窍。下面是一些涉及到的指标。

# 思考

研究的首要原则是思考、思考、再思考。毫不犹豫地将你的想法投入任何事情，这应该是第一件事。查找书籍或论文之前，请三思。通过思考解决的问题将会印象深刻，你可能会忘记想要查找的内容。思考不仅限于实验室，也可能会在洗澡或开车这样零散的时间，来思考科研的问题。 思考很有趣，如果你没有发现，那就说明科研并不适合你。

# 找到一个好榜样

你可以通过看榜样和推断来学习做研究，与你的导师进行的论文和讨论是达到这一目的资料来源。 找出一个在你的相关研究领域里很厉害的人，并了解他们的思维、研究和演讲风格。你可以通过仔细阅读其他论文来学习如何撰写论文，当你看到一个新问题时，问问你自己，以前类似的问题都问过哪些类型的问题，然后用这些问题来问新问题。坚持这样做下去，你应该能够推断得越来越远。不过你必须小心，不要走极端，在没有形成自己的观点之前就跟随别人的观点。

# 只管去做

有时，学习新事物的最好方法就是去做。你可以尝试新事物并从失败中不断学习，从而学到更多。不要害怕「失败」。如果实验证明你的假设是错误的，那不是失败，不要犹豫，进入实验室开始做实验。在学习滑冰或骑自行车之前，你一定会摔倒几次，而科研更糟糕。一旦你学会了如何骑自行车，就不会再跌倒了，但是在科研中，你永远不会停止失败，失败的次数要远远大于成功的次数。但是，正是从失败中学习，提高了你的理解力和成熟度。如果你发现科研过程很有趣，那么失败并不会令人生畏，实际上，失败会促使各种各样的发现。因此，你需要学会不关心失败，而要利用失败来获得新的想法。做科研有一点好处，就是你失败时没有人会知道，你只会写成功的论文。但这也具有欺骗性，因为你看不到别人的失败，但是失败总是在那里，不偏不倚。

# 理解和知识

理解你所知道的比知道很多更重要。成功的科学研究来自于对基础知识的深刻理解。当你读一篇论文时，问自己几个问题：如果在接下来的实验中改变方案会怎么样？为什么这是与众不同的？我能想出一个不同的方法吗？理解意味着超越眼前的能力，这意味着不仅要知道这里论述的内容，还要知道它如何适应更大的环境。

# 领导者

大多数学生会把他们的导师看作领导者，这是非常自然的反应，尤其是在科研的初期。但自己接手项目时，不用感到害怕。的确，导师拥有更多的经验，但是与导师相比，你花了更多的时间来思考科研项目，随时采取主动，为研究找到新的方向，这个习惯将在你的整个职业生涯中带来巨大的帮助。

# 读博期间学生的工作是什么？

你在这里进行自我培养并获得博士学位，学生有时将导师建议的目标或实验视为需要完成的「工作」，这可能适合某些实验。但是，这不是你完成博士学位的目标。你的导师事实上不要你做任何「工作」，你是为自己工作，而不是为导师工作。当导师提出实验的相关建议时，仅是你可能会采纳的建议，如果你不同意，请与导师进行讨论。

一些学生认为他们是完成导师科研项目的一种方式，然而并不是这样的。博士学位最重要的研究成果是学生，而不是论文。

# 读博期间导师的工作是什么？

就博士研究而言，导师的唯一工作就是指导学生。一些学生认为筹集资金来支持他们是导师的「工作」。这话有一半是对的，为研究提供资源（实验室、设备等）是导师和大学的工作。在某些系中，学生必须完全通过 TA 或 RA 来养活自己（与他们的博士学位无关）。为自己的博士研究设立一个RA，可以让你有更多的时间和自由进行研究。拥有这种自由是一种特权，在你的职业生涯中，任何时候都不会有这么多的自由、时间和资源用于你的智力发展，所以充分利用你在博士期间的时间。

大多数导师都非常努力地进行研究（以及教学和其他职责），他们在办公室，家里甚至在假期工作。他们中的许多人甚至以牺牲「家庭时间」为代价来工作。对导师而言，科研更多是一种生活方式而不是工作。尽管培养这种对科研的依恋需要时间，但导师们自然也会期待学生们的兴奋，真诚，责任和努力，即使是在博士研究生的早期阶段。

# 4. 与导师合作

学生和导师一起工作的方式因学生和导师的不同而不同。但这里有一个大概的轮廓。

# 读博期间

在大多数情况下，你和你的导师会在博士研究生早期就制定出明确定义的概括性目标。你的博士论文会分为三个阶段。

第一阶段的主要目的是向你介绍科研。这一阶段，导师通常会为你提供明确定义的目标（广义目标的子目标），如果成功完成，则可能发表高质量的出版物。这个开始阶段应该有明确的问题、目标和可交付的成果，这意味着学生应该清楚目标是什么，需要做什么。你和你的导师通常会共同努力，正确制定问题，解决问题，撰写解决方案，然后将其提交给合适的期刊和/或会议。

第二阶段的主要目标是巩固与研究相关的各个领域的技术技能（实验和/或理论）。在此阶段，导师将继续就目标和实验提出建议，范围从定义明确到模糊不清，但是在解决问题和编写解决方案方面，你将更多地依靠自己。在此阶段，你应该对实验/模型的细节有深入的了解，还应该能够自己合理地撰写论文，能够自己进行展示。

第三阶段的主要目标是增强你的科研「视野」。在这个阶段，你应该有足够的能力找到你自己的问题/实验以及答案。你还需要拓宽你对研究的观点，增强与其他相关研究领域的技能，能够理解其他相关领域的重要问题，完全熟悉发表论文的过程。你还应该确定你毕业后想做什么，并开始学习你尚未学过的相关技能。

# 与导师会面

大多数导师每周固定会与学生见一次面，你也可以在未安排的时间来拜访，或者发送电子邮件。一起工作很有趣，大多数导师都喜欢。大量的研究是自发的，源于我们的互动。你可能没有什么实质性的东西要在会议上汇报。没关系，只要不总是这样就行。

尝试尽可能多地交流，并且要坦诚。在你需要的时候征求反馈，你不仅可以自由地提出不同意见，而且被鼓励这样做。如果你不喜欢导师建议的某一组实验，那就说出来。如果你不明白为什么它是重要的，那就寻求动机。你越坦诚、越直接，你的进步就越大，告诉你的导师你需要他/她做什么。

为你的会议做准备。如果你计划提出或讨论某个方案或对某些数据进行分析，请首先自己考虑一下。准备好你的想法，这样你才能清楚地表达出来。如果你的导师不得不花更多的时间来理解你，你就会得到低质量的反馈，而且很难一起工作来解决问题。不要害怕提供细节，你的导师知道得越少，给出的建议质量就越低。

准备会议。如果你打算提出或讨论方案，或对某些数据进行分析，请首先考虑一下。 准备好你的想法，以便你清楚地表达它们。 如果你的导师需要花更多的时间来了解你，则你得到的质量反馈较低 ，因此很难共同寻求解决方案。不要害怕提供细节。你的导师了解得越少，建议的质量就越低。

把你和导师的讨论看作是一种练习，你可以把它展示给其他人和更多的听众。你的导师比其他人对你的发言更感兴趣，他/她会比其他人更关注你。如果他/她不明白你在说什么，那么可能没有人会明白。同样，如果你的演讲没有让你的导师兴奋，其他人也不会。

尝试弄清楚，在下一次会议之前，你的方向任务、问题、可交付成果（如果有）是什么？如果不确定是否会记住，请写下来。

最后，保持与导师的自由交流。如果你想知道你的导师在想什么，就询问他/她，不要玩猜谜游戏。记住，你和你的导师之间的关系不是雇佣型的。像任何其他关系一样，这种关系可能包含着兴奋、快乐、失望、不快和满足。当你和你的导师在某件事情上努力工作了很长一段时间后，你很难摆脱这些情绪。做到坦诚和善于沟通，你将能够很好地处理它。

# 5. 你能合格吗？

定期评估你的表现是有用的。最好的方法是问你的导师，衡量进步的方法有很多，直接输出，论文是最明显的。导师还会考察你对材料的理解和知识是如何呈现的。

参考上面的三个阶段的路径，在第一阶段结束时，最小的期望是你对自己的学科有很好的理解，了解其他人在这方面的工作。你应该知道，如果更改实验中的某些内容会发生什么，以及与其他实验相比是如何的。从开始研究到这个阶段，通常最多需要一年的时间。当你完成第二阶段的时候，你应该在材料中游刃有余了，并能够快速掌握新概念，并且做简单的事情，并在头脑中进行简单的证明/比较/分析。能够在不同的子区域之间建立联系，对什么是已知的和什么是未知的有一些感觉，并且熟悉基本的技术，工作中已产出了高质量的出版物。完成这一阶段所需的时间因人而异。在你完成第三阶段的时候，除了你对自己学科的熟练之外，你还应该有一种品味和成熟的感觉。完成这一阶段所需的时间也因人而异。

出版物对于顺利获得博士学位至关重要，出版物的数量随着你读博期间研究领域而不断变化。如果你想进入科研领域，无论是在实验室还是在学术界，他们都会查看你的出版物记录。请记住，仅凭获得博士学位不能保证一个好的工作或未来，我们都知道，杰出的大学辍学生比普通博士拥有更多的名望和金钱。你的目标应该是保持研究和论文的质量远高于平均博士学位的质量。在你所在的领域，出版物数量没有固定的规则，你应该尽可能保持出版物的高质量。除了论文之外，还希望你能提高书面和口头交流能力。记住，你可能需要与来自不同背景的人沟通，从科学家、专利律师到企业高管。

完成博士学位所需的时间取决于许多因素。成就是用产出来衡量的，而不是时间。为了便于讨论，我们只能讨论平均数。博士学位平均需要五年，在更短的时间里完成也是可行的。一个常见的误解是，导师想让学生呆很长的时间，这并不可信，一个顺利毕业的博士生也会对导师产生良好的影响。

# 6. 时间分配

人们在科研上分配时间的方式因人而异，每星期或每季度都有很大的差异。有些人在灵感或按截至日期的基础上工作，其他人则保持稳定的时间表。没有对错或好坏，找到最适合你的即可。

导师追求持续的生产力。优秀的博士生通常应在相当长的一段时间内表现出出色的生产力和进步，这包括提高理解力，自信心和成熟度以及看得见的产出输出（如论文）。

如果在某一天，甚至是某一周，你发现自己并不想做太多的科研工作，这是很正常的，时不时地休息一下。如果你不能集中注意力，度假可能比坐在实验室里强迫自己工作更好，如何在短期内分配时间取决于你自己。但是如果很长一段时间都没有显著的进展，你应该担心并和你的导师谈谈。如果你不知道自己是否取得了「足够的」进步，那就去问，不要害羞。

一般来说，博士生倾向于投入长时间到科研工作中，每周工作超过 70 小时并不罕见。事实上，很多导师花了很多时间在工作上。由于所需的时间因学科而异，所以没有什么神奇的数字可以用来确定一个目标。导师看重的是责任、真诚以及「我会竭尽所能把事情做好」的态度。
